Arduino Quadcopter

A guide to making your own inexpensive Quadcopter

The Build

Here’s an overview on how I built my Quadcopter Most Quadcopter are built using manual transmitter, receiver and flight controller. My goal is to make a ‘smart’ quad which can make minor decisions on its own. This is achieved using Arduino as a receiver and flight controller.

The main advantage is that the quadcopter has pre-programmed flight path. In addition, this allows for custom hardware integration such as webcam, sonar, etc.

Here’s a quick rundown on the parts needed.
PARTS FOR QUADCOPTER

MPU-6050 This chip provides gyro as well as accelerometer readings. Here is processing example on how to hooking up MPU-6050 with Arduino.

The first phase is controlled via standard Bluetooth connectivity. I used the MPU-6050 gyro and accelerometer sensor to determine the orientation of the quad. This still requires a lot of fine tuning and more testing to attain smooth and stable flight.
